## 🎯 Project Overview
The **Apex Iris Framework** is a high-performance system designed to stress-test classification models. It evaluates predictive integrity across clean datasets and corrupted feature sets. 

**Now Fully Automated:** The framework utilizes GitHub Actions to automatically train, evaluate, and rank submissions the moment a Pull Request is merged.

---

## 🛠️ Innovation: Dynamic CI/CD Evaluation
Unlike static leaderboards, our framework features an **Auto-Sync Engine**:
- **Automated Benchmarking:** Every PR triggers a virtual environment that runs the contributor's code against hidden test parameters.
- **Secure Handling:** Results are encrypted with **Fernet (AES-256)** and stored in unique user directories to ensure data integrity.
- **Instant Deployment:** The Live Leaderboard updates via GitHub Pages within seconds of a successful merge.

---

## 📊 Dataset & Perturbation Profile
Our framework utilizes a specialized binary-mapped version of the **Iris Botanical Dataset**, optimized for Graph Topology Ablation testing.

### ⚠️ The Ablation Challenge
Models are evaluated on two distinct test pipelines:
1. **Standard Set:** Clean feature vectors representing the "Ideal" topology.
2. **Perturbed Set:** Features injected with **Gaussian Noise** ($\mu=0, \sigma=0.1$) to simulate sensor degradation.

## 📥 Submission Protocol (New Automated Flow)
Follow these **4 steps** to enter the global rankings:

### 1️⃣ Fork the Repository
Click the **Fork** button to create your own copy of the infrastructure.
```powershell
git clone [https://github.com/](https://github.com/)[YOUR_USERNAME]/Apex-Iris-Framework.git

```
### 2️⃣ Develop Your baseline.py
Modify the provided script to implement your custom architecture.

*  **Requirement:** Ensure your script prints the final Accuracy and F1-Score to the console.
*  **Identity:** The system identifies you automatically via your GitHub Username.


### 3️⃣ Create a Pull Request
Commit your changes and open a PR to the main repository.

*  **Trigger:** Our GitHub Action will perform a "Dry Run" of your code to check for errors.


### 4️⃣ Merge & Rank
Once the PR is merged by the leads:

- The Apex Bot executes your model.
- Your metrics are extracted and encrypted.
- Your name and score appear on the Live Leaderboard instantly.
